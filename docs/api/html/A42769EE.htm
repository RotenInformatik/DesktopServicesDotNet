Remarks

The InstanceLocator is used in XAML to get instances from either an IServiceProvider (ServiceProvider) or a resolver function ( Resolver) and assign them to properties in XAML. For example, this can be used to retrieve and attach a view model to a DataContext in MVVM scenarios.

The instance to obtain can be either specified by its name, using the Name property, or its type, using the Type property. Name is treated as the string representation of the Type to use.

Instance resolving is done in the following order: If Resolver is not null, that function is used. If Resolver is null or returns null, ServiceProvider is used. set.
