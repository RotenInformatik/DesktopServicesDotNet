Remarks

A serial port on Windows is also called a COM port, e.g. COM1.

 Note
SerialPortInstance is not intended for actually accessing a serial port. SerialPortInstance is only used for managing the used/available ports. Use SerialPort to access a serial port. SerialPort instances can be created using CreateSerialPort.
