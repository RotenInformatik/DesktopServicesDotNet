Remarks

The common WPF controls which are supported as containers by this region adapter are: ContentControl, ItemsControl, Panel. All types derived from those are also supported.

Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
