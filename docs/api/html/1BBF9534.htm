Remarks

InvertedObjectBoolConverter is used when a reference value of any type needs to be used as boolean value.

Conversion from Object (the source) to Boolean (the target): If the value is null, the converter returns true. If the value is not null, the converter returns false.

Conversion from Boolean (the target) to Object (the source): If the value is false, the converter returns the parameter. If the parameter is an IEnumerable, its first element (or null if the sequence contains no elements) is returned. If the value is true, the converter returns null.
