SettingConversionMode Enumeration

Describes the setting conversion mode used by a ISettingConverter.

StringConversion0 The converter uses explicit conversion to/from strings, e.g. using ToString and TryParse methods. Therefore, only some particular types can be converted by the converter.
SerializationAsString1 The converter uses serialization to/from strings using some defined format (JSON, XML, etc.). Therefore, any serializable object can be converted by the converter (where serializability is defined by the converter itself).
