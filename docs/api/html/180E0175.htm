Remarks

This setting store is not read-only.

Sections in INI files are not supported. When reading, section headers are ignored. When writing, the file is emptied before the values are written without any sections (except WriteOnlyKnown is set, in which cases the structure of the file is preserved).

If the specified INI file does not exist, it will be created and no values will be read from it (because its empty after creation...).

Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
