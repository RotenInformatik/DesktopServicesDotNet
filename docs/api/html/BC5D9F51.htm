Remarks

RegionBinder is a convenience utility to work with regions (IRegionService) in MVVM scenarios. It defines an attached property (RegionNameProperty) which can be used to associate a container with a region (using the region services AddRegion(String, Object) method). It also defines an attached property (RegionServiceProperty) to specify the used IRegionService.

The used IRegionService is determined in the following order: If RegionServiceProperty is not null, that instance is used. If DefaultRegionService is not null, that instance is used. InstanceLocator is used to obtain an instance of IRegionService.

Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
