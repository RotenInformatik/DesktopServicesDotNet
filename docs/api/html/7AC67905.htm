Remarks

No indication (return value or exception) is provided to determine whether closing the connection suceeded.

 Important
Whether and how the connection is closed depends heavily on the configuration of Windows, the computer, the network, and the remote resource itself. It cannot be guaranteed that the connection is really being closed. Therefore, this method should only be seen as a "hint to Windows what to do".
